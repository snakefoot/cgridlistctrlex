Detailed Description

CGridColumnTraitCheckBox implements a checkbox as cell-editor.

It hacks the cell-image, and transforms it into a checkbox. To get/set the checkbox value of a cell use the methods GetCellImage()/SetCellImage() on CGridListCtrlEx (0 = Uncheck/1 = Check).
